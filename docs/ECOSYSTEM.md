# Ecosystem Overview

DumbNotes is a social note-taking and notebook platform inside the ZalomoLabs ecosystem.

High-level ecosystem parts:

- DumbNotes: posts, notebooks, profiles, comments, reactions, bug reports, and support flows
- Clubs: memberships, seats, entitlements, and ecosystem perks
- Store: checkout entry point and account dashboard
- MoveDrive storage bridge: optional storage visibility for Club users
- Admin tools: private tools for moderation, support, and operations
- AI agents: support tools for moderation, bug triage, summaries, and security review

This document is intentionally high-level and does not expose private infrastructure.
