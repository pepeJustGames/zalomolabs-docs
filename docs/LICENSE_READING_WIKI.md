# License Reading Wiki

Tags: `wiki` `license` `funny-but-useful` `read-before-copying`

This is the "something dumb of the day" license wiki.

It is a joke in tone. Do not take the jokes too seriously. But if you genuinely do not know how to read a license, then yes, take the license seriously.

## The 30-Second License Ritual

Before copying, reusing, reposting, or modifying something, ask:

1. Who owns it?
2. What am I allowed to do?
3. What am I not allowed to do?
4. Do I need attribution?
5. Can I use it commercially?
6. Can I modify it?
7. Does it apply to docs, code, assets, brand, or everything?

If your answer is "uhhh probably fine", that is not a legal strategy. That is a loading screen.

## Common License Words

| Word | Human meaning |
| --- | --- |
| `Allowed` | Things you can do without asking again |
| `Not allowed` | Things that get you bonked by reality |
| `Attribution` | Give credit |
| `Commercial use` | Using it to make money or support a product |
| `Derivative` | A thing based on another thing |
| `Warranty` | A promise that it works |
| `As-is` | It might explode metaphorically and that is your problem |

## DumbNotes Docs License In Plain English

For this public docs repo:

- You can read it.
- You can link to it.
- You can quote small parts with credit.
- You can use dummy config examples as templates.
- You cannot pretend to be ZalomoLabs.
- You cannot claim the brand.
- You cannot use docs as permission to access private systems.
- You cannot treat dummy keys like real access.

## Serious Part

Licenses are not decoration. They define permission.

If a file says documentation-only, that does not mean the private app source code is open-source.

If a config example says dummy, that does not mean you can hunt for real keys.

If a brand name is visible, that does not mean you can impersonate it.

## Mini Checklist Before Publishing Anything

- Did I remove real tokens?
- Did I remove private emails?
- Did I remove admin screenshots?
- Did I keep only public-safe wording?
- Did I link to the right license?
- Did I avoid promising things the app does not guarantee?

## Final Meme Rule

If you cannot explain the license to a tired friend in one minute, slow down and reread it.

Again: joke tone, serious permission. Tiny legal boss fight. Read the health bar.
