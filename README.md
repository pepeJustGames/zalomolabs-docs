# DumbNotes Public Documentation

This branch is the public documentation branch for DumbNotes and the ZalomoLabs ecosystem.

It is intentionally separate from the application source code. It contains public-safe information only:

- product overview
- Terms of Service
- Privacy Policy
- license notes
- security policy
- public configuration examples with dummy values
- high-level AI agents documentation

No real API keys, tokens, admin sessions, private user data, build artifacts, or backend secrets should be committed here.

## Products Covered

- DumbNotes
- ZalomoLabs Clubs
- ZalomoLabs Store
- MoveDrive-connected storage features
- Support and moderation tooling
